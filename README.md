# instawork-fullstack
## This is a project for instawork coop interview

**Version 1.0.0** 

## Installation

```bash
python -m pip install django
````

## Run this project
```bash
cd instawork-fullstack/hw2022
```

```bash
python manage.py runserver
```
Then this project will run on port 8000 at your localhost
